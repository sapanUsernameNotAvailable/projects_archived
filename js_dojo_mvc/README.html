Single page ajax web app

@author : sapan verma. This is a single page web app coded to exercise dojo(1.8) skills and to serve as a refrence implementation for any future projects which need to utilise the frameworks involved without much configuration hassle.

Frameworks used

  • spring (DI, MVC) : annotation based
  • hibernate 3.x: annotation base
  • dojo 1.8

web urls of interest :

  • / : base url for application
  • /mockingbird.htm : for application testing

highlights :

  • js architecture.
  • client - server communication via json.
